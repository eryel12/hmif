# hmif

Link Repository GitHub : 
https://github.com/eryel12/hmif.git

Tech Stack : 
- Bahasa : JavaScript
- Framework : ReactApp JS
- Library : react-router-dom, Flowbite-React, CSS
- Database : none (localStorage)

Struktur File (Hierarki) :
1. event-app, merupakan folder terbesar yang menyimpan semuanya
2. node_module, berisi library yang digunakan dalam web ini
3. public, berisi aset aset yang bersifat tidak berubah
4. src, merupakan bagian logika dan tampilan dari web
5. package.json, berisi daftar library dan command script
6. tailwind.config.js, berisi mode JIT, custom theme, dan file to react

Cara Instalasi :
1. pull repository dari GitHub
2. buka terminal dan ketik "cd event-app"
3. lalu ketik "npm install"
4. kemudaian ketik "npm start"
